# d-Briefs

**A curated gallery of legal case briefs.**

🌐 **Live site:** https://dbriefr.com  
📁 **Planning:** `p-dbriefs/`

---

## About

d-Briefs is a showcase for legal case briefs created with Casebriefr. Browse briefs by date, case name, or author.

## Structure

- `index.html` — Homepage
- `gallery.html` — Browse all briefs
- `brief-viewer.html` — View individual briefs
- `d-briefs-archive/` — Brief files (HTML/Markdown)
- `scripts/` — Gallery generation scripts
